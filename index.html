Compact CP Violation Experiment

Precision Measurement of δL in Neutral Kaon Decays

🌌 Physics Goal

We aim to measure the semileptonic charge asymmetry:

δL = [Γ(KL → π⁻ℓ⁺ν) - Γ(KL → π⁺ℓ⁻ν̄)] /
       [Γ(KL → π⁻ℓ⁺ν) + Γ(KL → π⁺ℓ⁻ν̄)]

This serves as a clean probe of CP violation in a compact, low-cost setup.

🔬 Beamline Design

  • Proton beam → K⁺ source
  • Charge-exchange in hydrogen-rich target → K⁰
  • Short-lived KS decays quickly, leaving KL beam

🛠️ Detector System

  • Dual-arm layout for charge identification
  • Trigger scintillators for event timing
  • Tracking chambers for momentum & vertex
  • Spectrometer magnet to separate π⁺/π⁻
  • Cherenkov + calorimeter for e⁺/e⁻ tagging
  • Optional muon filter for background suppression

💡 Innovation

We're designing a low-budget ($2000) Cherenkov-calorimeter module to identify electrons with sufficient resolution for a 10⁻³ level asymmetry measurement.

📁 Documentation

All calculations related to beam dynamics, detector response, decay kinematics, and background control are handled in our accompanying technical documents.
